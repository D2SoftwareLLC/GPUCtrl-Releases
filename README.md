# GPUCtrl Releases

This is the **public release distribution repo** for [GPUCtrl](https://gpuctrl.com)
— a commercial desktop application for managing and launching GPU-accelerated
applications with intelligent GPU routing.

The application source lives in a separate private repository. This repo exists
solely to host installer binaries and update metadata so the auto-updater and
the marketing site can resolve `releases/latest/download/...` URLs.

## Download

See the **[Releases](../../releases)** page on this repo, or visit
**[gpuctrl.com](https://gpuctrl.com)** for one-click download buttons.

Stable artifact filenames per release:

| Platform | Filename |
|----------|----------|
| Windows installer (NSIS) | `GPUCtrl-Setup.exe` |
| Windows installer (MSI)  | `GPUCtrl.msi` |
| Linux AppImage           | `GPUCtrl.AppImage` |
| Linux deb (Debian/Ubuntu)| `gpuctrl.deb` |

A `SHA256SUMS.txt` is published alongside each release.
